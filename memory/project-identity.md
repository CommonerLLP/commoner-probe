probe is **sousveillance infrastructure** for the state's mandatory disclosure systems
(parliamentary questions, budget allocations, committee proceedings, state assembly
records). It automates the move Ambedkar made with census blue books and Browne made with
Fanon's FBI file: seizing the state's own paperwork and turning it into evidence.

**Why:** CommonerLLP is an Ambedkarite project of data science counter-forensics. The
library must not describe itself using naturalising categories ("Indian government sites")
— that names the nation, not the obligation. Name the obligation: "state mandatory
disclosure infrastructure."

**How to apply:** Any README, docstring, or identity statement for probe/sansad-crawler
should use "state's mandatory disclosure" or "sousveillance infrastructure," not "Indian
government" or any geographic category.
